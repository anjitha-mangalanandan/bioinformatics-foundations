This repository documents my transition from biotechnology into bioinformatics through hands-on projects and structured self-study.

The goal of this 12-week learning portfolio is to build practical skills in:

- Linux command-line tools
- Bash scripting
- Python for bioinformatics
- R for data analysis
- Next-generation sequencing (NGS) workflows

Each week focuses on a specific skill required for building and understanding real-world bioinformatics pipelines such as RNA-seq analysis.
