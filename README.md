# bioinformatics-foundations
This repository documents my transition from biotechnology into bioinformatics through hands-on projects and structured self-study.
The goal of this 12-week learning portfolio is to build practical skills in Linux, Bash scripting, Python, R, and next-generation sequencing (NGS) data workflows, with the aim of developing and understanding bioinformatics pipelines such as RNA-seq analysis.
